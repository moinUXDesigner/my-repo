# PMS Application - Quick Start Guide

## 🚀 Getting Started

### Step 1: Login
1. Open the application - you'll see the login screen
2. Enter any username (e.g., `john.doe`)
3. Enter any password (e.g., `password123`)
4. Select a role:
   - **Employee** - For individual contributors
   - **Reporting Officer (RO)** - For team managers
   - **Reviewing Officer (RVO)** - For senior reviewers
   - **Accepting Authority (AA)** - For final approvers
   - **HRD Admin** - For HR administrators
5. Click **Sign In**

## 📋 Common Tasks

### Update Your Profile
1. Click your **avatar** (top-right)
2. Select **"My Profile"**
3. Click **"Edit Profile"**
4. Make changes
5. Click **"Save Changes"**
6. See success notification ✅

### Submit KRA Entry (Employee)
1. Navigate to **"KRA Entry"**
2. Fill in basic information
3. Add KPIs (ensure total weight = 100%)
4. Click **"Save Draft"** or **"Submit"**
5. See success notification ✅

### Review Subordinate (Officer)
1. Go to **"Team Dashboard"**
2. Toggle to **"Employee Level"**
3. Find the employee
4. Click **"Review"**
5. Complete evaluation
6. Submit for approval
7. See success notification ✅

### Generate Report
1. Navigate to **"Reports"**
2. Select report type OR use custom generator
3. Configure filters (optional)
4. Click **"Generate Report"**
5. See success notification ✅
6. Download from **"Recently Generated Reports"** table

### Download Report
1. Go to **"Reports"**
2. Scroll to **"Recently Generated Reports"**
3. Click **"Download"** on desired report
4. See download notification ✅

## 🆘 Troubleshooting

### Can't Login?
- Use any username and password (demo mode)
- Ensure you've selected a role
- Try refreshing the page

### Sidebar Not Showing?
- Click the menu icon (≡)
- Check screen size (auto-hides on mobile)
- Refresh the page

### No Notifications?
- Check top-right corner
- Notifications auto-dismiss after a few seconds
- Try the action again

### Can't Find a Page?
- Check your role permissions
- Some pages are role-specific
- Use sidebar navigation

### Mobile Issues?
- Ensure you're using a modern browser
- Check your screen orientation
- Try refreshing
